Go to the Diasend app and upload your latest data.
After uploaded, go to the 'View Data' - this takes you to https://international.diasend.com/patient/account/export-data
Export the Data
Type in the I'm not a Robot code, this will export to Excel
Open the spreadsheet, in Excel, Enable editing at the top and goto the 'Insulin use and Carbs' tab. File -> Save As, Comma Seperated File (.csv) that sheet only.
Ensure you close Excel down with the CSV file (otherwise this app will complain the file is already open)
Open this app, click GO. Load the CSV just created.
